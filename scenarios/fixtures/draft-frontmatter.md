# Onboarding Note

This is a short note that is intended to help explain the current state of the onboarding process. It is important to note that new employees are often given a lot of information across many different channels, which can be confusing and somewhat overwhelming.

There are three main things that should happen during the first week. First, the manager should explain what success looks like in the role. Second, the buddy should help the new person understand where things live. Third, the team should avoid scheduling meetings that are not actually necessary.
